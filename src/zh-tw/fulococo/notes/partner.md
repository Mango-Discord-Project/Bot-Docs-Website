在職業中可以招募夥伴幫助工作，夥伴會提供額外效果並且在工作結算後獲得經驗值，夥伴等級越高提供的效果越好。

### 夥伴總覽

| 夥伴     | 招募費用   | 額外效果                      | 升級材料         |
| -------- | ---------- | ----------------------------- | ---------------- |
| 亞梓莎   | 100,000 點 | 砍樹結算後木材基礎收穫+3      | 巧克力×1＋抹茶×1 |
| 芙蕾亞   | 100,000 點 | 釣魚結算後魚貨基礎收穫+3      | 鐵十字×1＋玩偶×1 |
| 利莉姆   | 100,000 點 | 採礦結算後礦石基礎收穫+3      | 星軌儀×1＋櫻花×1 |
| 梅露耶爾 | 100,000 點 | 額外10%機率釣到夢見鯉魚       | 巧克力×1＋櫻花×1 |
| 露絲     | 100,000 點 | 額外10%再觸發一次煉金         | 鐵十字×1＋抹茶×1 |
| 彌久里   | 100,000 點 | 損毀時30%機率保護工具不被損毀 | 星軌儀×1＋玩偶×1 |

::: tip 關於直接升級
每位夥伴升一級需要的材料都是固定的 3,000 沙丁魚＋300 鮪魚，可以透過 6 個夢見鯉魚直接兌換
1×夢見鯉魚 = 10×鮭魚 = 1,000×沙丁魚 或 100×鮪魚
:::
:::tip 關於升滿
升滿一位夥伴共需要 30,000 沙丁魚＋3,000 鮪魚，可以透過 60 個夢見鯉魚直接兌換；
升滿全部夥伴共需要 180,000 沙丁魚＋18,000 鮪魚，可以透過 360 個夢見鯉魚直接兌換。
:::

### 夥伴兌換

| 1x巧克力/鐵十字/星軌儀 | 300×鮪魚    |

## 指令相關

### `我的夥伴` / `夥伴 [夥伴名稱]`

查看自己的夥伴列表、招募狀態、目前同行

也可加上 `{夥伴名稱}` 查看特定夥伴的詳細資訊

### `招募 {夥伴名稱}`

花費 100,000 點招募夥伴，招募後需要選擇是否同行，同行後才會享受夥伴效果並且獲得經驗值

### `選擇同行 {夥伴名稱}`

選擇與某位夥伴同行，同行後才會享受夥伴效果並且獲得經驗值

### `選擇輔助 {夥伴名稱}`

選擇某位夥伴為輔助，輔助夥伴只提供被動效果，不提供主動技能

### `輔助切換`

切換主動夥伴與輔助夥伴的角色，主動夥伴提供主動技能與被動效果，輔助夥伴提供被動效果。

### `培養 {夥伴名稱} [次數]`

使用沙丁魚和鮪魚兌換特定的夥伴素材培養夥伴升級，預設培養一次，每次培養需要 3,000 沙丁魚和 300 鮪魚

### `裝備卡 {夥伴名稱} {卡片名稱}` / `卸下卡片 {夥伴名稱} {卡片名稱}`

裝備或卸下夥伴卡片，卡片提供額外效果

::: tip 關於新手裝備卡
可以透過 `@可可 兌換 旋轉門與手扶梯禮包 夢見鯉魚` 使用 10 個夢見鯉魚兌換新手裝備卡禮包，開啟後會拿到R到UR卡片，永久限購 1 個。

內容物：
- R卡：齒輪（攻擊 +40）
- SR卡：傳動框架（血量上限 +200）
- SSR卡：手扶梯（攻擊 +72｜血量上限 +360）
- UR卡：旋轉門（造成傷害時 +30～44｜受到傷害時減免 16～50）
:::

### `裝備武器 {夥伴名稱} {武器名稱}` / `卸下武器 {夥伴名稱}`

裝備或卸下夥伴武器，武器提供額外效果

::: tip 關於新手武器
可以透過 `@可可 兌換 試作新手武器禮包 夢見鯉魚` 使用 3 個夢見鯉魚兌換新手武器禮包，開啟後會獲得三個屬性的新手武器各一個，永久限購 1 個。

內容物：
- 1星試作小太刀
- 1星試作大剪刀
- 1星試作量子環
:::

### `使用物品 {物品名稱} [夥伴名稱]`

使用物品給夥伴，部分物品會提供額外效果

| 物品名稱  | 效果                              | 備註           |
| --------- | --------------------------------- | -------------- |
| 復活藥劑  | 使陣亡的夥伴復活並恢復 100% HP    | 需指定夥伴名稱 |
| 高喊水晶R | 下次討伐觸發克制前提升基礎數值    |                |
| 勘用水晶R | 下次夥伴受到傷害時減免 100 點傷害 |                |

### `夥伴數值` / `查詢 {玩家名稱} 夥伴數值`

查看自己的夥伴數值，或查看其他玩家的夥伴數值

### `挑戰 {玩家名稱}`

向其他玩家發起夥伴對戰，雙方的主動夥伴將會進行對戰，勝利的一方將獲得 1×夢見鯉魚。
