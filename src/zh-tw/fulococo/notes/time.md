### 時段

一天分成 4 個時段，與[園藝](https://dcbot.ytmango.uk/zh-tw/fulococo/notes/job.html#%E5%9C%92%E8%97%9D%E5%B8%AB)有關。

查詢當前時段： `@可可 時段`

| 上/下半日 | 時段     | 期間         | 附註                       |
| -------- |-------- | ------------- | -------------------------- |
| 上半日    | 凌晨    | 00:00 ~ 05:59 | 不可打劫、詞條「凌晨的」生效 |
| 上半日    | 早上    | 06:00 ~ 11:59 |                            |
| 下半日    | 下午    | 12:00 ~ 17:59 |                            |
| 下半日    | 晚上    | 18:00 ~ 23:59 |                            |

06:00 ~ 17:59 砍樹 / 採礦 / 海釣 1% 機率獲得 日暮果實 / 日暮礦 / 日暮魚

18:00 ~ 05:59 砍樹 / 採礦 / 海釣 1% 機率獲得 月光果實 / 月光礦 / 月光魚
