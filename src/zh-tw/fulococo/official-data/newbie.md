【芙蘿可可 Bot｜新手教學】 `@可可 新手教學`

第一天建議照這個順序：

1. `@可可 簽到`  
   每天 00:00 與 12:00 各刷新一次。

2. `@可可 打卡`  
   每個整點可打卡一次，是早期點數與資源來源。
   :::tip
   使用 `@可可 謎題 Maimai音遊` 回答問題可以快速獲得點數
   :::

3. `@可可 註冊工作`  
   花費 10,000 夢見點數取得工作證，解鎖等級、職業、採集與背包玩法。

4. `@可可 購買 石製斧頭 / 石製鎬子 / 劣質釣竿`  
   先買基礎工具，再依職業進行砍樹、採礦、海釣。
   :::tip
   建議直接購買鐵製斧頭/鐵製鎬子/優質釣竿，合計 60,000 點
   ::: 

5. @可可 轉職 伐木師 / 挖礦師 / 釣魚師 / 魔法師 / 神匠師  
   職業決定你能做哪些工作。

6. `@可可 背包`  
   查看自己的點數、等級、工具、水晶、重要素材與夥伴狀態。

7. `@可可 攻略`  
   打開[攻略站](https://dcbot.ytmango.uk/zh-tw/fulococo/)查完整資料。

常用查詢：
- `@可可 指令查詢 每日`
- `@可可 指令查詢 工作`
- `@可可 功能 夥伴`
- `@可可 價格 全部`

……先不要一口氣學完。能簽到、打卡、註冊工作，就已經踏進門了。
