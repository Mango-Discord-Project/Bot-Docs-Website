:::note 2026/05/28 23:23

| 項目               | 規則                             |
| ------------------ | -------------------------------- |
| 是否需要玩家魔法師 | 不需要                           |
| 附魔者             | 可可                             |
| 點數消耗           | 普通附魔價格 ×110%               |
| 低星成功率         | 1→2、2→3、3→4、4→5 成功率減半    |
| 高星成功率         | 5→6 之後維持普通附魔成功率       |
| 損壞保護           | 會自動使用保護器抵銷一次工具損壞 |
| 冷卻               | 1 分鐘                           |

| 觸發情境                           | 效果                                |
| ---------------------------------- | ----------------------------------- |
| 砍樹、採礦、海釣時觸發 1% 工具損壞 | 若有保護器，會自動消耗 1 個抵銷損壞 |
| 高星附魔失敗並觸發工具損壞         | 若有保護器，會自動消耗 1 個抵銷損壞 |
| 沒有保護器                         | 工具會直接損壞並消失                |

:::
