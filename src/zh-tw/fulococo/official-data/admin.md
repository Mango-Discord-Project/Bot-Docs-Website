::: note 2026/05/28 18:10

| 功能           | 指令範例                            | 備註               |
| -------------- | ----------------------------------- | ------------------ |
| 給點數         | `@可可 給點數 @對方 1000`           | 管理員限定         |
| 扣點數         | `@可可 扣點數 @對方 500`            | 管理員限定         |
| 減少點數       | `@可可 減少點數 @對方 500`          | 管理員限定         |
| 移除點數       | `@可可 移除點數 @對方 500`          | 管理員限定         |
| 修改連續簽到   | `@可可 修改 @對方 連續簽到天數 7`   | 管理員限定         |
| 修改簽到總次數 | `@可可 修改 @對方 簽到總次數 20`    | 管理員限定         |
| 修改打卡總次數 | `@可可 修改 @對方 打卡總次數 30`    | 管理員限定         |
| 中止謎題       | `@可可 答題中止`                    | 管理員或出題者可用 |
| 版本更新公告   | `@可可 版本更新` / `@可可 更新公告` | 管理員公告用       |
| 休眠調整公告   | `@可可 休眠調整` / `@可可 休眠公告` | 管理員公告用       |
| 故障重啟公告   | `@可可 故障重啟` / `@可可 重啟公告` | 管理員公告用       |
| 開機公告       | `@可可 開機公告` / `@可可 可可開機` | 管理員公告用       |

:::
